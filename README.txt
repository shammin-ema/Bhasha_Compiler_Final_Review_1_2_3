============================================================
Project: Bhasha Compiler - Review 2 Final Version
============================================================

Main Review 2 Statement:
The compiler is implemented in Java and generates executable
Python target code in output.py.

============================================================
CORE COMPILER OVERVIEW
============================================================

Implementation Language:
- Java

Target Language:
- Python

Generated Target File:
- output.py

Compiler Workflow:
Source Program (.bhasha)
        ↓
Lexical Analysis
        ↓
Parsing + AST Creation
        ↓
Semantic Analysis
        ↓
Three Address Code Generation
        ↓
Python Code Generation
        ↓
Executable Python Program

============================================================
GENERATED OUTPUT FILES
============================================================

- output.py
    Generated executable Python program

- output.txt
    Compiler execution log for valid source programs

- error_output.txt
    Compiler execution log for invalid/error source programs

- program_output.txt
    Final output generated by the Python target program

============================================================
AUTOMATION SUPPORT
============================================================

run.bat
- Compiles Java files
- Executes compiler
- Runs generated Python program
- Saves generated output automatically

run_error.bat
- Executes error demonstration source program
- Demonstrates syntax and semantic error handling

============================================================
WHY IF-ELSE IS EXCLUDED
============================================================

According to Review 2 requirements,
code generation is implemented for all non-if-else features.

Conditional statements (if-else) are reserved for:
- Review 3
